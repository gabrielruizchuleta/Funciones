Etiquetas Extras

Etiqueta PRE

La etiqueta pre sirve para respetar los espacios y líneas en blanco al contrario de la etiqueta párrafo

Hola soy gabo tengo 17años

            Hola  soy  
            gabo     tengo     17años

Etiqueta abr

La Etiqueta que se utiliza para las abreviaturas

Según datos de la OMSes el consumo excesivo de azúcar provoca daños severos a la salud de los niños

Etiqueta bdo

Si se desea cambiar la dirreción de un texto , se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Vivo muy a prisa, la vida de artista Otra más para la lista.Por: Hassan Emilio Kabande Laija

Etiqueta small

La etiqueta small hace el texto más pequeño

Este texto está dentro de una etiqueta small

Etiqueta Sub

Si se desea subir un subíndice, se debe utilizar la etiqueta sub

1433

Etiqueta sub

Etiqueta time

Para escribir fecha y horas se debe utilizar la etiqueta time

Mañana es dia 6 de Noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirreción utiliza la etiqueta adress

Loma dorada No.100 Colonia Lomas de Rosales CP: 89100 Tampico, Tamaulipas
